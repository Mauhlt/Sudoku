# Sudoku Solver:
## Features:
1. Solve sudoku board 
- check that start position is valid 
- check that solution passes checks
- solve a easy, medium, hard problem
2. Optical Character Recognition 
- convert an image of a sudoku board to 9x9 board of values 
- pass inputs into solver to solve board 
3. Mobile / PC solver 
- take a picture with app 
- pass picture to solver
